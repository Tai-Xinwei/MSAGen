Overview

A4SFramework aims to reduces redundant coding work for researchers and facilitates large distributed training.

  1. In the User-friendly Interface, examples are provided for quickly setting up a training loop using our trainers.

  2. In the Large Distributed Training, examples show how to implement pipeline parallelism and tensor parallelism through clear examples.

  3. In the Installation, packages will be released by pip in the future.

  4. In the Model Zoo, examples show how to use models in A4SFramework.

  5. In the Data Pipeline, data pipeline will be developed.

  6. In the Training Args, the detailed info of training args in A4SFramework will be provided.
